使用指南
1、在src/views下创建文件夹，在文件夹下创建index.html,index.css,index.ts,这个文件夹名字会被转为对应的html的名字
2、在app.js中加入上一步创建的文件夹名字。 如下 entry = [ common('文件夹名字'，‘首页文件的中文名字’) ]
3、启动项目   npm  run   start
4、生成静态文件  npm  run  build

// 开发中新建的的页面需要走上面的 1、2、3、步

功能点：
1、es6编译，sass编译
2、css、js自动加入到对应的html中加上了hash  防止缓存
3、html，css，js 变化是页面自动刷新
4、js，css 压缩
5、自动生成一个简单的导航页面index.html
